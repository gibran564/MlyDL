#### Practica 1 — Data Wrangling

Preparación, limpieza y transformación de un conjunto de datos real como paso previo al entrenamiento de modelos de Machine Learning. Se abordan técnicas de manejo de valores nulos, normalización, codificación de variables categóricas y análisis exploratorio.

## Proyecto Final — Sistema de Diagnostico Asistido por IA en Imagenes Medicas

El proyecto final consiste en el desarrollo de un **sistema de clasificación y análisis de radiografías** mediante técnicas de Deep Learning, con el objetivo de asistir en el diagnóstico médico a través del reconocimiento de patrones en imágenes de rayos X.

El sistema aplicará redes neuronales convolucionales (CNN) entrenadas sobre datasets de imágenes médicas para identificar anomalías o condiciones clínicas relevantes, integrando los conocimientos adquiridos a lo largo del curso.

| Fase | Descripción | Estado |
|------|-------------|--------|
| Fundamentación | Marco teórico, contextual y diagnóstico del problema | ⬜ |
| Planeación | Diseño del sistema, selección de dataset y arquitectura del modelo | ⬜ |
| Ejecución | Entrenamiento, validación y ajuste del modelo CNN | ⬜ |
| Evaluación | Métricas de desempeño, conclusiones y áreas de mejora | ⬜ |
